## Overview

This **client-side** program currently includes features as follows:

- Account Support
  - Sign up
  - Sign in
  - Recover account
  - Delete account 
  - Show account information
- Email
  - Send & fetch emails
  - Save & load drafts
- Communication 
  - Save & load contacts
